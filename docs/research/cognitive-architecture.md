# Cognitive Architecture Research

## Research Areas

### Memory Systems
How memory should be stored, recalled, reinforced and forgotten.

### Emotional Context
Representing emotional states and their impact on decisions.

### Reflection
Mechanisms allowing an agent to evaluate previous actions.

### Meta Reasoning
Reasoning about reasoning.

### Collective Intelligence
Groups of agents collaborating and sharing knowledge.

## Open Questions

- How should memory decay occur?
- How can emotional context be represented?
- Can identities evolve?
- How should memories be ranked?
