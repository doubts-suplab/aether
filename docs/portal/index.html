See the Aether ecosystem actually work

Aether is a distributed cognitive computing platform — persistent memory, governed agents, knowledge, and human-in-the-loop workflows. This portal is a hands-on walkthrough: each platform below has a small, faithful simulation of its real core loop, running entirely in your browser. Nothing here calls a server — it's a teaching model of the mechanics, not the production runtime.

How to use this: open any platform below and interact with the controls — move the confidence slider, recall a memory, run a RAG query, approve a workflow step. Watch the decision envelopes, memory strengths, and state transitions update live. The How it fits page shows how the five platforms compose into one cognitive fabric.

What ties them together

One rule you'll see everywhere

Aether's central governance invariant appears in every simulation:

confidence < 0.8  →  never auto-enforce  →  route to a human

The agent proposes; the runtime disposes. No agent approves its own consequential decision — that guarantee is structural, not a convention. Grid enforces it at the gate; Flow raises the human task; Core records the outcome.

Bounded contexts, clean seams

Each platform owns exactly one capability and never reaches into another's store:

Corepersonal memory & cognition
Gridagent execution & governance
Memoryshared team memory & federation
Vaultdocuments, knowledge & RAG
Flowworkflow & human approval

These simulations are illustrative teaching models with simplified, deterministic logic — real thresholds and domain rules, but no LLM, database, or network. They exist to make the mechanics tangible, offline.
